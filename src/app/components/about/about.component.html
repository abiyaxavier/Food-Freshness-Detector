About the project

Why teach a machine to smell trouble?

An estimated third of all food produced globally is wasted — often because spoilage is caught too late, or perfectly good produce is thrown out on suspicion alone. FreshEye explores whether a lightweight vision model can make that judgment call faster and more consistently than the human eye, right from a phone camera.

Why AI is useful here

Freshness shows up visually — color shifts, texture breakdown, dark spotting — before it's obvious in smell or touch. A convolutional neural network trained on thousands of labeled examples can pick up on these subtle visual cues consistently, at scale, and instantly, which is exactly the kind of pattern-recognition task deep learning excels at.

The dataset

The model is trained on a labeled Fresh/Rotten image dataset (~3,000 images) spanning apples, bananas, oranges, tomatoes, mangoes, potatoes, and other common produce — split 80% training, 10% validation, and 10% testing to fairly measure real-world accuracy.

Project objectives

  • Build an end-to-end image classification pipeline using transfer learning.
  • Serve real-time predictions through a REST API built with FastAPI.
  • Deliver a responsive, user-friendly Angular interface for uploads and results.
  • Document the full workflow as a learning resource for future students.
